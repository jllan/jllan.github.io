一 从预编译版本或从仓库安装pcl

1. 安装pcl

  • 使用官方提供的预编译版本

    地址
    http://www.pointclouds.org/downloads/linux.html

问题
sudo apt-get install libpcl-all时有依赖问题无法解决

  • 从deepin软件仓库直接安装

此方法安装成功

2. 安装python-pcl

地址
https://github.com/strawlab/python-pcl

问题

python setup.py build_ext –inplace

pkg-config pcl_common-1.7
pkg-config pcl_common-1.6
pkg-config pcl_common
Makefile: 5: recipe for target ‘pcl/_pcl.so’ failed
make: * [pcl/_pcl.so] Error 1
