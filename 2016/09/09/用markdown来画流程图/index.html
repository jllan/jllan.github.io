定义元素的语法

  • tag就是元素名字，

  • type是这个元素的类型，有6中类型，分别为：

    • start # 开始
    • end # 结束
    • operation # 操作
    • subroutine # 子程序
    • condition # 条件
    • inputoutput # 输入或产出

  • content就是在框框中要写的内容，注意type后的冒号与文本之间一定要有个空格。

  • url是一个连接，与框框中的文本相绑定
